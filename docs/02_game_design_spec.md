# ゲーム仕様書: Clockwork Descent

## 概要

Clockwork Descent は、時計塔の地下へ潜るデッキ構築型ローグライクである。

プレイヤーは各戦闘でカードを使い、戦闘後にデッキを強化し、分岐マップを進む。ランの途中で得たカード、遺物、イベント結果によって戦略が変化する。

## 一文コンセプト

「壊れかけの時計塔を下りながら、時間を消費して強力な行動を選ぶターン制デッキ構築ゲーム」

## 体験の柱

1. 毎ターン、限られたエネルギーで最善手を考える
2. デッキが少しずつ壊れた機械のように噛み合っていく
3. 短期的な生存と長期的なビルド成長の選択に悩む
4. 敵の意図が見えるため、理不尽ではなく読み合いで負ける

## 対象プラットフォーム

初期想定:

- PC
- マウス操作
- キーボードショートカットは後回し

## ラン構造

1ランは3層で構成する。

| 層 | テーマ | 通常敵 | エリート | ボス |
| --- | --- | ---: | ---: | ---: |
| Act 1 | 錆びた機関室 | 5種 | 2種 | 1種 |
| Act 2 | 歯車庭園 | 6種 | 2種 | 1種 |
| Act 3 | 中枢時計室 | 6種 | 3種 | 1種 |

縦切り版では Act 1 のみ作る。

## 基本リソース

| リソース | 説明 | 初期値 |
| --- | --- | ---: |
| HP | 0になるとラン終了 | 75 |
| Energy | カード使用に必要。ターン開始時に回復 | 3 |
| Block | そのターン中のダメージを軽減。ターン終了時に0 | 0 |
| Time | 一部カードで増減する特殊リソース | 0 |
| Gold | ショップやイベントで使用 | 99 |

## カード領域

| 領域 | 説明 |
| --- | --- |
| Draw Pile | 山札 |
| Hand | 手札 |
| Discard Pile | 捨て札 |
| Exhaust Pile | 廃棄されたカード |

ターン開始時、プレイヤーは5枚引く。山札が足りない場合、捨て札をシャッフルして山札にする。

## ターン進行

### 戦闘開始

1. デッキをシャッフルして山札にする
2. 敵グループを生成する
3. 初期手札を引く
4. 敵の初回行動意図を決定する

### プレイヤーターン

1. ターン開始効果を処理する
2. Energyを最大値まで回復する
3. カードを5枚まで引く
4. プレイヤーが任意の順でカードを使う
5. ターン終了ボタンを押す
6. 手札を捨て札に送る

### 敵ターン

1. 敵が表示済みの意図に従って行動する
2. 状態異常のターン終了処理を行う
3. 次ターンの意図を決定する

## カード分類

| 種別 | 説明 |
| --- | --- |
| Attack | 主にダメージを与える |
| Skill | 防御、ドロー、補助 |
| Gear | 継続効果。商用作品でいうパワー枠に近いが、名称は本作固有 |
| Status | 敵やイベントで混入する邪魔カード |
| Curse | 長期的なデメリットカード |

## キーワード

| キーワード | 説明 |
| --- | --- |
| Exhaust | 使用後、戦闘中は戻らない |
| Retain | ターン終了時に手札に残る |
| Overwind | Timeを得る代わりに次ターンEnergyを失う |
| Rewind | 捨て札からカードを戻す |
| Pierce | Blockを一部無視する |
| Tune | ランダム対象ではなく、指定したカードを強化候補にする |

## 状態異常

| 状態 | 対象 | 効果 |
| --- | --- | --- |
| Weak | 双方 | 与える攻撃ダメージが25%低下 |
| Fragile | 双方 | 得るBlockが25%低下 |
| Burn | 双方 | ターン終了時に値ぶんHPを失う。毎ターン1減る |
| Slow | 双方 | 次に使うカードのコストが1増える |
| Precision | プレイヤー | 次のAttackが追加ダメージを与える |

## 初期キャラクター

名称: Warden

役割:

- 防御を積みながら、Timeを使って強いターンを作る
- 初心者向けに、単純な攻撃と防御の比率を高める

初期デッキ:

| カード | 枚数 |
| --- | ---: |
| Strike Gear | 5 |
| Guard Plate | 4 |
| Clock Jab | 1 |

初期ステータス:

| 項目 | 値 |
| --- | ---: |
| Max HP | 75 |
| Starting Gold | 99 |
| Starting Energy | 3 |

## マップノード

| ノード | 説明 | Act 1 出現比率 |
| --- | --- | ---: |
| Monster | 通常戦闘 | 45% |
| Elite | 強敵戦闘 | 10% |
| Rest | 回復または強化 | 12% |
| Shop | 購入、削除 | 10% |
| Event | ランダムイベント | 18% |
| Treasure | 遺物獲得 | 5% |

## 報酬

通常戦闘後:

- カード報酬3択
- Gold
- 低確率で遺物

エリート戦闘後:

- カード報酬3択
- Gold
- 遺物1個確定

ボス戦闘後:

- ボス遺物
- 次Actへ進む
